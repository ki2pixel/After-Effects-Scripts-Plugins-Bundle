# After Effects – Documentation Technique Interne

**TL;DR** : Ce dossier contient les patterns et techniques ExtendScript avancés pour le développement production-ready dans Adobe After Effects.

## Pour qui ?

Développeurs ExtendScript intermédiaires à avancés qui ont besoin de patterns robustes pour Shape Layers, interfaces ScriptUI et manipulation avancée des propriétés AE.

## Organisation

Ce dossier regroupe la connaissance technique essentielle pour trois domaines critiques du développement ExtendScript moderne. Chaque document aborde un aspect spécifique de l'automatisation AE avec des patterns testés en production.

Les guides fondamentaux se concentrent sur l'architecture hiérarchique des Shape Layers, la création d'interfaces dockables robustes et la cartographie complète des MatchNames AE. Cette organisation permet aux développeurs de maîtriser rapidement les patterns réutilisables plutôt que de réinventer des solutions pour chaque projet.

## Navigation recommandée

Commencez par l'architecture Shape Layers si vous débutez avec les patterns modernes. Pour les projets d'interface, passez directement aux guides ScriptUI. Utilisez la cartographie MatchNames comme référence continue lors de l'automatisation des propriétés AE.

## Compléments

Les scripts concrets dans Scripts_AE/ illustrent ces patterns en action. La documentation officielle Adobe reste la référence API complète, tandis que PyShiftAE offre une alternative Python moderne pour les nouveaux projets.
