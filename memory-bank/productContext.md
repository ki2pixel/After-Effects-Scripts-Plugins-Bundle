# Product Context

## Project Overview
**Project Name**: After Effects Scripts & Plugins Bundle
**Type**: Collection of Adobe After Effects automation tools and scripts
**Date Created**: 2025-02-06

## Project Scope
- Collection of automation scripts for Adobe After Effects
- Python-based automation via PyShiftAE
- ExtendScript/JSX tools for AE scripting
- Plugin development and integration

## Components
- **Scripts_AE/**: Main collection of AE scripts (559 items)
- **PyShiftAE/**: Python automation framework
- **PyShiftBridge/**: Bridge for Python-AE integration
- **AETK-main/**: After Effects Toolkit
- **docs/**: Documentation and guides

## Organization
- Root level contains main script collections
- Documentation organized in docs/ directory
- Development tools and frameworks in dedicated subdirectories

## Standards
- Follow Adobe After Effects scripting conventions
- Python 3.11+ compatibility for PyShiftAE
- ExtendScript/JSX for legacy AE scripting
- Modular architecture for script organization
