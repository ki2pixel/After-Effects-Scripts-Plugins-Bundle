

Installation:

1. Copy the "com.aescripts.proio" folder into the following folder:

Win : C:\Program Files (x86)\Common Files\Adobe\CEP\extensions\
Mac : /Library/Application Support/Adobe/CEP/extensions

2. Apply:

Win - Add Keys.reg
Mac - install-as-admin

3. You should allow scripts to write files in After Effects:

CS - CC 2018  Edit->Preferences->General->Allow Scripts to Write Files and Access Network

CC 2019 - 2020 Edit->Preferences->Scripting & Expressions->Allow Scripts to Write Files and Access Network
   
   
   
Установка:

1. Скопируйте папку "com.aescripts.proio" в следующее место в каталоге установки AE:

Win : C:\Program Files (x86)\Common Files\Adobe\CEP\extensions\
Mac : /Library/Application Support/Adobe/CEP/extensions

2. Применить:

Win - Add Keys.reg
Mac - install-as-admin

3. Поставьте галочку в:

CS - CC 2018  Edit->Preferences->General->Allow Scripts to Write Files and Access Network

CC 2019 - 2020 Edit->Preferences->Scripting & Expressions->Allow Scripts to Write Files and Access Network
