1. Copy the jsxbin file to the following location:
Windows:
C: \ Program Files \ Adobe \ Adobe After Effects "corresponds to your AE version" \ Support Files \ Scripts \ ScriptUI Panels \ (docking panels)
MAC:
Master HD / Applications / Adobe After Effects "corresponds to your AE version" / Scripts / ScriptUI Panels / (docking panels)
2. Open AE, in order to easily access the network so that your test your script is registered. With this option enabled in After Effects' Preferences> General tab
Win: Edit> Preferences> General> Check * Allow Scripts to Write Files and Access Network
Mac: After Effects> Preferences> General> Check * Allow Scripts to Write Files and Access Network
