COMPATIBILITY

Adobe Premiere Pro CC 2017 - CC 2020

INSTALLATION
============

ZXP Install:
- Follow the "ZXP Installer" link in this folder to download and install the ZXP Installer from aescripts ("https://aescripts.com/learn/zxp-installer/")
- Open the installer, and drag and drop the "com.aescripts.proio.zxp" file in this folder to install Pro IO
- Restart Premiere Pro and After Effects to use Pro IO

Windows Manual Installation:
- Rename "com.aescripts.proio.zxp" to "com.aescripts.proio.zip"
- Right click and select "Extract All" to unzip the archive
- Move the "com.aescripts.proio" folder into this directory: "C:\Program Files (x86)\Common Files\Adobe\CEP\extensions"
- Restart Premiere Pro and After Effects to use Pro IO

Mac Manual Installation:
- Rename "com.aescripts.proio.zxp" to "com.aescripts.proio.zip"
- Double click to unzip the archive
- Move the "com.aescripts.proio" folder into this directory: "/Library/Application Support/Adobe/CEP/extensions/"
- Restart Premiere Pro and After Effects to use Pro IO

FEATURES
========

Watch Bins
– Add new watch bins to your project from real directories.
- Customize the watch bin settings to to include, flatten, or ignore subfolders, and recognize 
  image sequences.
– Sync watch bins whenever you have new files in your directories to import any new files.
- Unlink watch bins as needed while still keeping imported assets.
– Add Watch Bins to you project templates so all future projects are ready to sync.
- Edit the "Allowed File Name Endings" file to controll which footage types are imported into 
  your project.

Export Presets
– Create new Export Presets for commonly used settings with EPR files for Media Encoder or After Effects 
  presets for the AE render queue.
– Customize your export presets with the desired format settings, render settings, destination, 
  subfolder preference, and naming convention.
– Your top three exports are easily accessed with the dedicated buttons. To access any others export 
  presets, right click and reorder.
- Share your export preset file to use your presets on other systems.

KNOWN LIMITATIONS
=================
- In Premiere, image sequence frame rate will revert to default settings if "Refresh Image Sequences" is enabled.
- Premiere CC 2018 and up will export project panel selection if selected sequences within 10 seconds, otherwise 
  exports the active sequence.
- Premiere Pro CC 2017 cannot unlink and sync watch bins from project panel selection or export from 
  project panel selection. Must modify watch bins from settings panel and export active sequence.
