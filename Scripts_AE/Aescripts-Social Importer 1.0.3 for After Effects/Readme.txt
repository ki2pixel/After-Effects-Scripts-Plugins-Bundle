
Installation:

Copy the "Social Importer" folder into the following folder:

Win : C:\Program Files (x86)\Common Files\Adobe\CEP\extensions\
Mac : /Library/Application Support/Adobe/CEP/extensions

Apply:

Win - Add Keys.reg
Mac - install-as-admin

You should allow scripts to write files in After Effects:
CS - CC 2018  Edit->Preferences->General->Allow Scripts to Write Files and Access Network

CC2019  Edit->Preferences->Scripting & Expressions->Allow Scripts to Write Files and Access Network
   


Установка:

Скопируйте папку "Social Importer" в следующее место в каталоге установки AE:

Win : C:\Program Files (x86)\Common Files\Adobe\CEP\extensions\
Mac : /Library/Application Support/Adobe/CEP/extensions

Применить:

Win - Add Keys.reg
Mac - install-as-admin

Поставьте галочку в Edit->Preferences->General->Allow Scripts to Write Files and Access Network
