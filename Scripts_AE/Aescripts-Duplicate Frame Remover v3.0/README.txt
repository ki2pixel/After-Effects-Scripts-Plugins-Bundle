 INSTALLATION

(1) copy the file DuplicateFrameRemover.jsxbin into the folder
"Support Files/Scripts/ScriptUI Panels" (on Windows) or
"Scripts/ScriptUI Panels" (on Mac) of your After Effects
installation. 

(2) Start DuplicateFrameRemover (via the "Window" menu in After Effects)

ABOUT

Automatically detects duplicate frames in footage
