【Script Installation】

1. Copy the loopmaster.jsxbin file to the AE script directory:

Win: ... Adobe After Effects CC \ Support Files \ Scripts \ Scriptui Panels \
MAC: Application \ Adobe after Effects CC \ Scripts \ Scriptui Panels \


2. Open the AE software settings, check the above*allow script read and write and access network:
Win: Edit-Preferences-Conventional
MAC: After Effects CC-Preferences-Conventional

AE CC2019.1 or higher version is:
Win: Edit-Preferences-Scripts and Express
MAC: After Effects CC-Preferences-Script and expression


3. You can see the loopmaster.jsxbin script at the top of the software at the top of the software
